BasePaper

구글 앱 스토어, 애플 앱 스토어 등에서 화면 미리보기 영역에서 주로 사용되는 플리킹 플러그인입니다. 자바스크립트와 jQuery 동시에 지원하며 시작위치 지정, 슬라이더의 페이징 기능등 많이 사용되어지는 기능을 넣었습니다.

(IE7, IE8 등 css3를 지원하지 않는 브라우저에서는 사용할 수 없습니다.)

How to Use

Method

toPrev 이전 슬라이더 이동
toNext 다음 슬라이더 이동
toSlide( index ) 지정된 슬라이더 이동
getPos 현재 슬라이더 인덱스
startSlideShow 슬라이더쇼 멈춤
stopSlideShow 슬라이더쇼 시작

Option

elem required. Type: Selector or jQuery or Element
바깥 감싸는 DOM Element
list required. Type: Selector or jQuery or Element
슬라이더를 감싸는 DOM Element(ex: ul, dt 태그)
node required. Type: Selector or jQuery or Element
슬라이더 DOM Elements(ex: li, dd 태그)
speed Type: Integer (default: 400)
슬라이더간 움직인 시간
range Type: Integer (default:10)
사용자 터치시, 터치 시작부터 끝날때 까지 거리가 일정 거리 이상이면 다음(혹은 이전) 슬라이더로 넘어갈때의 비율(백분율)
loop Type: Boolean (default: true)
루프 여부. false로 설정되면 마지막(처음) 슬라이더에서 처음(마지막) 슬라이더로 이동 안함
slideShowTime Type: Boolean or Integer (default: 3000)
슬라이더쇼 시간
snap Type: Boolean (default: true)
snap 기능
create Type: Function
생성시 콜백 함수
active Type: Function
슬라이더 이동 후 콜백 함수
